## Functionalities

Add tasks <br>
Complete task by clicking on it <br>
Delete task by clicking on icon <br>
Drag and drop tasks

## Running the project

Clone this repository :
```bash
https://github.com/sujacquin/todo-client.git
cd todo-client
```

Start the server
```bash
npm start
```
Build the projet
```bash
npm run build
```
